                                                         TIC TAC TOE GAME

README 

Tic Tac Toe is a board game designed for two players.
The objective in the game its to connect 3 marks of a single type in the same row, column or diagonal.In order to introduce
the mark either 'X' or 'O' the user will have to wait for his trun an introduce the index number where he wants to stablish his mark 
The visual studio program can be found in git and can be executed with double click on it.
for more detailed information about the game access to the next link https://www.exploratorium.edu/brain_explorer/tictactoe.html
